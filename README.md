# Kabinet24

Landing page for [kabinet24.by](https://kabinet24.by) — hourly room and office rental service in Minsk, Belarus.

Лендинг для [kabinet24.by](https://kabinet24.by) — сервис почасовой аренды кабинетов и залов в Минске.

## Stack / Стек

- Single-page PHP/HTML site / Одностраничный сайт на PHP/HTML
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Deployment / Деплой

Deployed via cPanel Git integration — push to `master` triggers automatic deployment.

Деплой через cPanel Git — пуш в `master` запускает автоматическое развёртывание.
